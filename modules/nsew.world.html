encode_area (chunk_map, start_x, start_y, end_x, end_y, encoded_area)
Encode the given rectangle of the world The rectangle defined by {start_x, start_y, end_x, end_y} must not exceed 256 in width or height.

Parameters:

  • chunk_map
  • start_x int coordinate
  • start_y int coordinate
  • end_x int coordinate
  • end_y int coordinate
  • encoded_area EncodedArea memory to use, if nil this function allocates its own memory

Returns:

    returns an EncodedArea or nil if the area could not be encoded

See also:

    decode
